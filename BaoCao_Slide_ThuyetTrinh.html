🌳 Skill Tree — 3 nhánh — 15 node

Movement
Walk Speed → Run Speed → Air Control → Dash → Wall Run → Double Jump + Grapple
+Stamina mỗi node | SP: 2→3→5→8→12
Aim
Recoil → Crit 10% → Crit 20% → Crit x1.5 → One-shot Crook + Bonus Special
+Damage mỗi node | SP: 2→3→5→8→12
Intelligence
XP Radius → x1.1 → Radius +10 → x1.15 → Radius +15 + Highlight
+HP mỗi node | SP: 2→3→5→8→12

🏆 GAME OVER & ACHIEVEMENT

Thống kê đầy đủ + 5 thành tựu

💀 Game Over

  • Story mode: Chapter, quest hoàn thành, journal, score
  • Wave mode: Score, wave, kill, best score
  • 3 lựa chọn: Restart (checkpoint), Main Menu, Quit

🏅 5 Achievement

  • Speedrunner — Hoàn thành 5 chương <11 phút
  • Hell Slayer — 130 Crook kills
  • At Ease, Cooper — 20 kills khi wall run
  • Tank Slayer — Hạ Tank đầu tiên
  • Close Call — Trong bán kính 5m khi Boomer nổ

Persistent — lưu local + sync PlayFab
